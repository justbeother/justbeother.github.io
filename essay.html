В качестве хостинга для сайта я выбрал Github по нескольким причинам. Во-первых, он бесплатный. Во-вторых, в нем нет всей этой мишуры из стандартных "красивых" шаблонов, как на других бесплатных хостингах, как Ucoz. В-третьих, в ближайшем будущем я планирую использовать Github для коммитов своих проектов, а это неплохая возможность получше узнать его инфраструктуру.
